1. 请认真阅读[《MSMP基本规则》](./rules/basic)[《MSMP对作弊的界定》](./rules/cheat)。
2. [下载或配置MFC](./download)，运行start.cmd并在左上角创建本地账号然后启动游戏
3. 根据鹅群公告中的MSMP服务器地址添加服务器，如进入服务器即默认你**已经认真阅读并理解了所有规则**，之后如出现违规，处理时管理员不会接受任何“什么规则”“忘记了”“我是傻逼”一类解释。按照聊天框中服务器的指引设置密码，即可开始游玩

:::caution

请给自己设置一个自己不会忘记同时有一定复杂度的密码并防止账号被盗，如因玩家个人原因账号被盗，maincore_tech.不会负任何责任

:::
