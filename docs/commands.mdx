通过mc内置的或者插件扩展的指令，你可以在游戏中更方便的办到一些事情。下面只列出了一部分，具体可以自行搜索
```
/back  //回到自己死亡的位置
/tpa <玩家名>  //请求传送到指定玩家身边
/tpahere <玩家名>  //请求指定玩家传送到自己身边
/tpcancel  //取消传送请求
/tpaccept  //同意传送请求
/tpayes  //同上
/tpdeny  //拒绝传送请求
/tpano  //同上
/sethome <家的名字>  //设置当前位置为<家的名字>，每个玩家最多只能设定1个位置
/home <家的名字>  //传送到设定的家位置
/delhome <家的名字>  //删除设定的家位置
/skins  //打开皮肤选择菜单，只能选择现有皮肤
/skin url <皮肤url>  //更改自己的皮肤，皮肤url为png文件的下载直链（自定义皮肤可上传至图床（如imgse.com）后获取url）
```

补充几个原版可用的指令：

```
/msg <玩家>  //悄悄的对指定玩家说话，只有指定玩家能看见
```
