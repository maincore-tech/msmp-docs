**若无特别声明，所有注册服务器及加入服务器的玩家均适用于这一规则**

- 禁止任何形式的小号（op账号除外），禁止登录他人账号，发现即永ban
- 禁止任何形式的作弊及不公平游戏，发现永ban。请先阅读理解[《MSMP关于作弊的界定》](./cheat)并根据[如何进服](../how-to-enter)自行设置标准客户端，之后进入游戏。
- 禁止以任何形式影响玩家正常游戏（击杀登录中的玩家，末地门复制TNT卡服，破坏末地门，卡出生点刷人头，禁人塔，禁人书等）
- 禁止以任何形式刻意造成服务器卡顿（末地蘑菇，更新抑制器等）
- 允许使用的刷物品机器必须有完善的销毁与搜集装置，严禁实体过多造成卡服炸服，严禁放生红石飞行器
- 禁止任何游戏外流通货币交易游戏内物品，私下交易造成后果服务器不承担任何责任
- 遵守中华人民共和国法律
- 除以上几条外，游玩时还应适当遵守公序良俗

## 注意这些
- 除以上行为外不做限制，在游戏中击杀玩家、破坏他人成果、盗用他人物品均不违反规则。玩家自行承担后果
- 不论游戏中发生什么事，都请在游戏中解决，禁止任何超出游戏的恶意行为，情节严重者永ban
## 关于刷物品
- 禁止利用特性复制物品
- 允许利用原版特性刷特定物品，包括铁轨、地毯、线等
## 保证
- 本服以玩家为主，为公益娱乐性质。赞助不能改变你的游戏体验，但可以获得一些来自maincore_tech.的服务（具体可以商议）
