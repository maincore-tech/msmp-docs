title: OP相关规定
sidebar_position: 3
sidebar_label: OP相关规定
---

- 所有OP应以其他玩家体验为先。若出现问题，OP应第一时间尽力解决，否则请向受影响玩家说明情况
- 若需在游戏内进行操作，请以OP账号进入游戏。保证账号安全，绝不出借、转让、交易OP账号。不能无故对单个玩家使用指令或修改游戏内容
- 任何情况下
```
/kill
/tp
/ban
```
等等指令只应对严重违反规则的玩家使用。进行tp不能泄露或记录该玩家坐标。传送到正常玩家身边应使用`/tps`
- 保密世界种子，不能将种子用于查找结构等行为
- 若你从普通玩家成为了OP（即TU（TrustedUser/授信用户））不得将此身份用于炫耀、区别地位等。OP没有也不应有优越性~~事实上这是一件十分麻烦的工作~~
- TU由一个或以上原始OP或三个或以上TU认证，可被同等程序撤销。仅原始OP有权操作服务器系统与冻结任意用户，事后必须公示原因。TU权限只能用于服务器面板与游戏内终端
- TU需要与原始OP线下会见才能认证成为原始OP。实际上如果有需要我们会自己去找你帮我们忙
- TU认证相关详询管理员
- 不得影响普通玩家游戏对等性、公平性
- 可以应邀监督pvp或某个疑似作弊的玩家
- 不可将特有权利用于窥探他人隐私、其他阵营情况
